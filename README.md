🧠 ArogyaAI – Smart AI Health Ecosystem
Predictive • Preventive • Personalized Healthcare using AI, IoT & Genomics
📌 Overview
ArogyaAI is a next-generation health platform that combines AI, Computer Vision, Smart Analytics, IoT sensors, and Genomic Intelligence to deliver real-time, personalized health insights. It integrates a web + mobile application with a hardware-based sensor system to enable proactive and data-driven healthcare.
🚀 Features
📸 AI Food Recognition – Detect food from images & estimate calories
🍛 Indian Food Intelligence – Supports mixed meals & local dishes
🧠 Smart Nutrition Analysis – Macro breakdown (protein, carbs, fats)
🤖 AI Diet Assistant (RAG-based) – Personalized diet guidance & myth-busting
🏃 Fitness Tracking – Activity monitoring + exercise recommendations
🧍 Posture Detection – Real-time posture correction & rep counting
⚙️ IoT Sensor Integration – Tracks environmental & health parameters
🧬 Genomic Prediction – Predicts potential disease risks
📊 Interactive Dashboard – Visual insights of health & progress

⚙️ Installation & Setup
# Clone the repository
git clone https://github.com/your-username/ArogyaAI.git

# Navigate to project folder
cd ArogyaAI

# Create virtual environment
python -m venv venv
source venv/bin/activate   # (Linux/Mac)
venv\Scripts\activate      # (Windows)

# Install dependencies
pip install -r requirements.txt

# Run backend server
uvicorn main:app --reload
